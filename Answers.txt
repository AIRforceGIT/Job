2. Assembler - программирование на уровне регистров. 
   С++ - низкоуровненвый язык, подходит для работы с "тяжелыми" программами из-за возможностей эффективного управления памятью.
   (создание ОС, драйверов, игр, возможно даже бэк-енд сервера)
   С# - намного более user-friendly нежели плюсЫ. Язык высокого уровня, подходит для автоматизации, тестирования,
   да и вообще для всего(а значит не для чего)
   Python - наиболее популярный сейчас язык. высокий уровень. Machine learning, big data, написание быстрых script`ов,
   серверов (django например) и вообще из-за огромного кол-ва библиотек для всего(действительно для всего)
   Html, Css, javascript- front-end. Html - статические объекты страниц, css - чтобы все было красиво, javascript - интерактивность страниц
   PHP - back-end

3. Компилятор - встроенная программа сборщик решения. В некоторых языках его нет, потому что есть Интерпретатор.
При компиляции все блоки транслируются в низкоуровневый код(ассемблер, машинный код), который и выполняет машина.
(перед выполнением компилятор анализирует и транслирует программу целиком). Интерпретатор построчно анализирует и выполняет исходный код программы.

4.http и https - протоколы клиент-серверного обмена, http - не защищен и устарел, https - топ. Поддерживает шифрование, что и обеспечивает безопасность.
